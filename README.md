- Cross platform
- Easy and intuitive UI

## Installation

Install project with npm

```bash
  git clone [url]
```

After cloning the repository

```bash
  cd fe/blind-tooth-fe
```

Install dependecies
```bash
  npm install
```

To run project 
```bash
  npm run start
```
